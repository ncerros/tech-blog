# Installation

Before connecting to the database it is necessary to install package.jon. 
1. Run npm i
2. Install dependencies
    bcrypt
    connect-session-sequelize
    dotenv
    express,
    express-handlebars
    express-session
    mysql2
    sequelize
3. Install dev dependencies
    jest
    nodemon
